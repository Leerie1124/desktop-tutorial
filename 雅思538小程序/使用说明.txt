苹果手机使用方法：

1. 先把“雅思538小程序”这个文件夹放到一个能用 Safari 打开的地址。
   - 最稳妥：上传到 GitHub Pages、Netlify、Vercel 或自己的 HTTPS 网站。
   - 也可以电脑和手机同一 Wi-Fi，用本地服务器临时打开。

2. 用 iPhone 的 Safari 打开 index.html 对应的网址。

3. 点 Safari 底部“分享”按钮，选择“添加到主屏幕”。

4. 回到桌面，点“雅思538”图标即可像小程序一样全屏使用。

本地临时预览命令：

在这个文件夹里运行：
python -m http.server 8080

然后用 iPhone Safari 打开：
http://电脑局域网IP:8080/

注意：直接把 html 文件发到微信或文件 App 里也能打开，但不能完整获得“添加到主屏幕”和离线缓存效果。
